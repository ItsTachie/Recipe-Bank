# 🍳 Welcome to My Recipe Bank

Welcome to my personal digital cookbook! This project started as a way to expand my cooking repertoire beyond my usual staples, tracking everything from traditional Zimbabwean comfort foods like Sadza to global flavors from the Middle East and Europe.

### 💻 About This Project
I wanted a highly efficient way to document my recipes. Now with my recipes documented I don't struggle to pick what I'm cooking during the week and I have the ingredients I need to get. The site is hosted publicly so I can access them from anywhere. This site is:

* **Written** locally in [Obsidian](https://obsidian.md/) using Markdown. - I already use Obsidian for my uni note taking so it just made sense. Obsidian also allows for templating which makes it really easy to create new recipes
* **Built** using Python and the [MkDocs](https://www.mkdocs.org/) static site generator (using the mkdocs `material` theme).
* **Hosted** automatically via GitHub Pages.
